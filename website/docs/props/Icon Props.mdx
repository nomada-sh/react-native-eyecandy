id: iconprops
title: Icon Props
---

Component to render as icon, can be a React Component (e.g. `Icon`), or a React Element (e.g. `<Icon />`).

When passing a React Component:

```jsx
<IconButton icon={Plus} />
```

or

```jsx
<IconButton
  icon={({ size, color, style }) => (
    <Plus size={size} stroke={stroke} style={style} {...otherProps} />
  )}
/>
```

You can use any component as an icon, regardless of color, size, or style. However, some props may be more suited for an icon to render nicely inside a certain component.

When passing a React Element:

```jsx
<TextInput iconRight={<Plus size={32} {...otherProps} />} />
```
